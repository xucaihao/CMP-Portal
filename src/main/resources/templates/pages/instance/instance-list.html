登录云主机

运行 RDP 文件方式登录

    下载 RDP 文件，运行后将唤起本地远程桌面程序，需要开启远程登录端口(TCP:3389)

    Linux 和 Mac OS X 系统请先安装远程桌面程序

标准登录方式

  • 使用 Windows 系统的电脑

    1、点击 开始菜单>>Run，输入 “mstsc”命令，打开远程桌面对话框

    2、输入框内输入 云主机公网IP，点击“连接”

    3、输入用户名： Administrator、[该CVM实例的密码]，点击“确定”

  • 使用 Linux 系统的电脑

    1、请安装对应版本的远程桌面连接程序，推荐使用rdesktop

    2、用户名： Administrator， CVM实例： 云主机公网IP

  • 使用 Mac OS X 系统的电脑

    1、请下载安装微软针对Mac OSX版 远程桌面链接程序

    2、输入用户名： Administrator、[该CVM实例的密码]，点击“确定”

浏览器 VNC 方式登录

    暂不支持复制粘贴、中文输入法。 立即登录

    提示：VNC 方式登录仅在首次登录时验证密码，请务必开启 MFA 二次验证提高安全保障级别

重置密码

您所选的 1 个实例 将执行重置密码操作，您是否确认操作？
12到16位字符，至少包括三项（[a-z],[A-Z],[0-9]和[()`~!@#$%^&*-+=_|{}[]:;'<>,.?/]的特殊符号），且不得包含用户名，如：“administrator”（大小写不敏感）
两次输入的密码不一致，请重新输入一致的密码。

提示：重置实例密码后，只有在控制台重启实例才能生效。
